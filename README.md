## 👨‍🏫 프로젝트 소개
3D 방치형 게임으로 자동으로 스테이지를 진행하는 게임입니다.
1인 개발

## 💜 주요기능

- 디자인 패턴
  - Singleton Pattern
  - Observer Pattern
  - State Pattern
  - Factory Pattern
- Object Pool 

- NavMesh를 이용한 적 AI(FSM) 구현
- Scriptable Object를 이용한 데이터 구현
- 확장 메서드
- 시드를 이용한 절차적 맵 생성
